Код реализует генетический алгоритм для аппроксимации полиномом.
1. Селекция производится в виде турнира на 10 родителей
2. Мутация работает только в случае повторения результатов (она минимальная, поэтому программа корректно работает только с мелкими числами)
3. Для красивого графика результата работы точки X брались последовательно, но их можно ввести и случайно
